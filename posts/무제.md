## 서식 참고 (작성 시 삭제)

### 제목

```
# 제목 1 (H1)
## 제목 2 (H2)
### 제목 3 (H3)
#### 제목 4 (H4)
```

### 텍스트 강조

```
**굵게**
*기울임*
~~취소선~~
`인라인 코드`
```

<mark>형광펜 하이라이트</mark> → `<mark>텍스트</mark>`

### 구분선

```
---
```
---
### 목록

```
- 항목 1
- 항목 2
  - 하위 항목

1. 순서 있는 항목
2. 두 번째
```
- 항목 1
- 항목 2
  - 하위 항목

1. 순서 있는 항목
2. 두 번째
### 표

```
| 열1 | 열2 | 열3 |
|-----|-----|-----|
| 내용 | 내용 | 내용 |
```

| 열1  | 열2  | 열3  |     |     |
| --- | --- | --- | --- | --- |
| 내용  | 내용  | 내용  |     |     |
### 인용

```
> 인용문 텍스트
```

> 이렇게 보입니다
